navis tools

Global Flags:

    -debug

    -verbose

    -out-directory:


Building Navis:

    -module:navis to build navis as module.

    -collection:navis add a module collection named as 'navis' with navis directory path.

    -navis-build-mode: shared(default), embedded.
        in shared mode the module will be linked with navis shared library,
        in other hand embedded mode will compile navis into your module or launcher binary.


Building Modules:

    -module:"output_name"="path/to/module/directory"

Building Launchers:

    -module:"output_name"="path/to/launcher/directory"

Building Shaders:

    -shader:"output_name"="path/to/my_shader.sc"
        specify 'path/to/my_shader.sc' and it will look for 'path/to/my_shader.sc.vs' for vertex shader and look for 'path/to/my_shader.sc.fs' for fragment shader.

    -clear-shader-modules
        clear all modules generated for each type and profile.

    -no-opengl
        will not compile opengl shaders.

    -no-opengl-es
        will not compile opengl es shaders.

    -no-direct3d
        will not compile direct3d shaders.

    -no-metal
        will not compile metal shaders.

    -no-vulkan
        will not compile spirv shaders.

    -no-gnm
        will not compile pssl shaders.
